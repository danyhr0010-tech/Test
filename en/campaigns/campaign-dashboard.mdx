<br />

The Campaign Dashboard provides a real-time view of a campaign run's progress, including aggregate statistics, per-contact results, and data export options.

## Accessing the dashboard

Click the **Dashboard** action on any campaign row that has been executed. The dashboard loads the most recent run by default.

## Header cards

The top section displays four summary cards:

| Card | Content |
|---|---|
| **Run selector** | Dropdown to switch between different runs of the same campaign. Includes a reload button (2-minute cooldown). |
| **Campaign info** | Campaign name and ID (click to copy). |
| **Run date** | Date and time the selected run was executed. |
| **Registers** | Total register count and phone count. Click **"More details"** to see max simultaneous calls and trunk type configuration. |

## Register table

Below the statistics, a detailed table lists every contact in the run:

| Column | Description |
|---|---|
| **Register ID** | Unique identifier for the contact (sortable) |
| **To Phone** | Target phone number |
| **Status** | Outcome status badge |

### Status filter

Use the status dropdown at the top of the table to filter registers by one or more outcome statuses. The filter options are populated dynamically from the run data.

### Inspecting a register

Click on a register row to view its details:

| Tab | Content |
|---|---|
| **Params** | The input parameters sent to the bot for this contact (from the CSV columns) |
| **Results** | The conversation results showing the bot flow path — node name, classification, and user transcription |

## Call list panel

For voice campaigns, the dashboard includes a call list management panel:

| Element | Description |
|---|---|
| **Call list status** | Current status of each call list chunk |
| **Stop** | Halt the active call list |
| **Recover** | Resume a stopped call list |
| **Download** | Export results as CSV |

## CSV download

From the dashboard you can export run results as a CSV file:

1. Click the **Download** button.
2. Optionally filter by status or state values before exporting.
3. The modal shows the filtered quantity and unique contact count.
4. Click **Download** to generate and save the CSV.

The exported file includes the contact phone numbers and input parameters, named with the campaign name and timestamp.
