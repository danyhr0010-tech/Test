## Overview

Mintlify allows you to customize your documentation site to match your brand identity.

## Color Customization

You can customize your site's color scheme by editing your configuration file.

### Primary Colors

Define your brand colors:

```json
{
  "colors": {
    "primary": "#0D9373",
    "light": "#07C983",
    "dark": "#0D9373"
  }
}
```

## Branding

### Logo

Add your company logo to display in the navigation bar.

### Favicon

Set a custom favicon for your documentation site.

## Styling Options

- **Theme**: Choose between light and dark mode
- **Typography**: Customize fonts and text styles
- **Layout**: Adjust spacing and component styling

<Tip>
Changes to styling will be reflected after pushing to GitHub.
</Tip>
