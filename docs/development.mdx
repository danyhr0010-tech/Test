## Prerequisites

- Node.js (v16 or higher)
- Mintlify CLI

## Setup

Install the Mintlify CLI:

```bash
npm i -g mintlify
```

## Preview Changes

Run the development server:

```bash
mintlify dev
```

Open [http://localhost:3000](http://localhost:3000) to preview your documentation.

## Workflow

1. Edit `.mdx` files or update `openapi.json`
2. Changes auto-reload in your browser
3. Push to GitHub when ready
4. Mintlify automatically deploys your changes

<Note>
Changes to `openapi.json` will automatically update the API reference documentation.
</Note>
