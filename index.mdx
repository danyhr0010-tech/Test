# Math API Documentation

Welcome to the Math API documentation. This API provides mathematical operations and statistical calculations.

## Overview

Math API is a REST API built with FastAPI that offers:

- **Calculator Operations**: Basic mathematical operations
  - Addition, subtraction, multiplication, division
  - Power/exponentiation
- **Statistics Operations**: Statistical calculations
  - Mean (average)
  - Median
  - Standard deviation

## Features

- **REST API**: 10 fully documented endpoints
- **OpenAPI Specification**: Auto-generated from Python code
- **Type Validation**: Request/response validation with Pydantic
- **Interactive Documentation**: Built-in Swagger UI
- **Testing**: 34 unit tests with 100% pass rate
